CNC Preview Handler
===================
Interactive 3D toolpath preview for .nc .gcode .gc .g .tap .cnc files.

INSTALL: Right-click install.ps1 -> Run as Administrator
UNINSTALL: Right-click uninstall.ps1 -> Run as Administrator

CONTROLS
  Left-drag      Orbit
  Right-drag     Pan
  Scroll         Zoom
  Double-click   Reset view

COLOURS
  Blue      Rapid moves (G0)
  Orange    Cutting moves (G1)
  Green     Arc moves (G2/G3)
